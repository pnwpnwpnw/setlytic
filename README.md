# Setlytics

Static marketing site for Setlytics — "Less busywork, more business."

## Pages

- `index.html` — Home
- `services.html` — Services
- `about.html` — About
- `contact.html` — Contact

## Run locally

No build step required. Serve the folder with any static file server, e.g.:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000

## Deployment

Hosted on GitHub Pages from the `main` branch (root): https://pnwpnwpnw.github.io/setlytic/
